Fases del Hacking

1. Fase de reconocimiento

En esta fase, el hacker se centra en la recoleccion de información sobre el objetivo que se busca atacar. Esto puede incluir direcciones IP, sistemas operativos y configuraciones de seguridad del sistema informatico de su empresa. Estos datos se pueden obtener de manera pasiva, al investigar datos públicos como sus sitios web o redes sociales, o tambien de manera activa, lo que implica obtener los datos por parte de terceros o tecnicas de rastreo mas invasivas.

2. Fase de escaneo

En esta fase se profundizan los conocimientos sobre el objetivo. Esto implica que, mediante el escaneo, el hacker se encarga de identificar los dispositivos, puertos, programas, servicios que se ejecutan. Ademas, investiga las vulnerabilidades.

3. Fase de acceso

En esta etapa, con los datos ya obtenidos, el hacker procede a simular ataques a los sistemas en un entorno controlado, para ingresar en ellos las vulnerabilidades identificadas. Aqui es donde comenzará a poner a prueba la capacidad defensiva de los sistemas ante ataques como inyecciones SQL, ataques XSS, etc.

4. Fase de mantenimiento de acceso

Una vez el hacker ya haya vulnerado el sistema, el trabajo consiste en mantener ese acceso el mayor tiempo posible, con el fin de realizar multiples ataques y asi obtener hackear el sistema. Para seguir dentro del objetivo infectado, el hacker puede hacer uso de malware para controlar los sistemas y equipos sin ser detectado.

5. Fase de borrado de huellas

Finalmente, la tarea final del hacker consiste en eliminar cualquier rastro de su presencia en el objetivo que se ha atacado (computadoras, sitios web, redes). Para esto puede ocultar su direccion IP, eliminar archivos de registro o utilizar conexiones cifradas.
